# 50.017-Graphics-and-Visualisation-Project

## SUTD Graphics and Visualisation (50.017) Project

![Morph and Explode Gif](./public/data/morphNexplode.gif)

The goal of our project is to to create a particle system that morphs from one 3D shape to another, and exploding in the process. We are doing it because it looks cool.

## Viewing the deployed app

The deployed app can be found [here](https://khaizon.github.io/50.017-Graphics-and-Visualisation-Project/)

## Setup Instructions

### Install Dependencies

```
npm i
```

### Start development

```
npm run dev
```
